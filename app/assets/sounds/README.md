# Audio Assets for ZenOne

This folder should contain breathing phase sounds:

## Required Files

| File | Phase | Description |
|------|-------|-------------|
| `inhale.mp3` | Inhale | Rising, calming tone (~4 sec) |
| `exhale.mp3` | Exhale | Descending, relaxing tone (~6 sec) |
| `hold.mp3` | Hold | Gentle sustained hum (~4 sec) |
| `ambient.mp3` | Background | Soft ambient pad (looping) |
| `chime.mp3` | Session end | Completion chime |

## Recommended Sources (Royalty-Free)

- [Freesound.org](https://freesound.org) - Search "meditation bells", "ambient pad"
- [Pixabay](https://pixabay.com/music/) - "Relaxation" category
- [Zapsplat](https://www.zapsplat.com) - SFX library

## Audio Specs

- Format: MP3 or M4A
- Sample Rate: 44.1 kHz
- Channels: Stereo (for binaural beats)
- Bitrate: 128-192 kbps

## Generating Tones

You can generate pure tones with Audacity:
1. Generate → Tone → Sine wave
2. Apply fade in/out
3. Add reverb for ambiance
4. Export as MP3
